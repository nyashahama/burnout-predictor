# Action Engine Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transform the burnout predictor from a passive scoreboard into an action coach with a closed daily loop: check in → get action plan → do action → see result.

**Architecture:** Backend-first approach: add DB column and API fields first, then wire frontend components that already exist (BurnoutAlert, RecoveryPlan, PersonalizedInsight, follow-up system), then build new UI (score-tiered action plan, streak dots, consistency metric, expanded check-in, feedback loop). The backend follow-up system already works — the frontend just needs to call it.

**Tech Stack:** Go (backend), Next.js/React (frontend), PostgreSQL (DB), sqlc (DB queries), Tailwind/shadcn (UI), Resend (email — existing)

- [ ] **Step 5: Add `FindSmallWinsImpact` to score package**

Create `backend/internal/score/smallwins.go`:

```go
package score

import (
	"strings"
)

// FindSmallWinsImpact checks how many times a small-wins keyword appeared in
// past notes and how many points the next-day score improved on average.
// Returns the average improvement in points (0 if insufficient data).
func FindSmallWinsImpact(entries []NoteEntry, smallWins string) int {
	if len(entries) < 5 || smallWins == "" {
		return 0
	}

	keywords := strings.Fields(strings.ToLower(smallWins))
	var totalImprovement int
	var matchCount int

	for i := 0; i < len(entries)-1; i++ {
		if entries[i].Note == "" || entries[i+1].NextScore == nil {
			continue
		}
		note := strings.ToLower(entries[i].Note)
		for _, kw := range keywords {
			if strings.Contains(note, kw) {
				delta := entries[i].Score - *entries[i+1].NextScore
				if delta > 0 {
					totalImprovement += delta
					matchCount++
				}
				break
			}
		}
	}

	if matchCount < 2 {
		return 0
	}
	avg := float64(totalImprovement) / float64(matchCount)
	if avg < 3 {
		return 0
	}
	return int(avg)
}
```

Note: The `NoteEntry` type already exists in the score package (`score.NoteEntry` has `Score int`, `Note string`, `NextScore *int`).

	}

		ID:     fu.ID,
		UserID: user.ID,
	}); err != nil {
		respond.Error(w, http.StatusInternalServerError, "failed to dismiss follow-up")
		return
	}

	respond.JSON(w, http.StatusOK, map[string]string{"status": "dismissed"})
}
